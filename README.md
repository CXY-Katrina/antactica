# 南极 Reveal.js Project

一个基于 reveal.js 3.9.2 的单页演示项目。

## Preview

```bash
python3 -m http.server 8000
```

然后打开 <http://localhost:8000>。

## Style

- 背景色使用 Pantone 2020 Color of the Year Classic Blue 的常用网页近似色 `#0F4C81`。
- 中文字体统一为 `Microsoft YaHei` / `微软雅黑`，并带有系统中文字体回退。
- 英文字体统一为 `Consolas`，并带有等宽字体回退。
